## 1. 项目说明

本仓库包含问卷数据的交叉分析、卡方检验，以及多张可视化图表的绘制脚本。  
所有脚本均使用 Python 编写，主要依赖：`pandas / numpy / matplotlib / scipy / openpyxl`。

---

## 2. 文件结构与功能说明

| 文件名 | 作用 |
|---|---|
| `analysis.py` | 对“二手平台问卷”的 Excel 数据做交叉分析（列联表）+ 卡方检验 + 输出并保存图表 |
| `cagr.py` | 中国二手市场规模相关的复合年均增长率（CAGR）计算 + 趋势图保存 |
| `motivate.py` | 横向条形图（barh）：展示“二手交易动机（多选）”各选项占比 |
| `age.py` | 柱状图：展示二手交易用户年龄段占比 |
| `shentou.py` | 双轴组合图：用户规模（柱状）+ 渗透率（折线）趋势对比 |
| `Z.py` | 环形图（doughnut chart）：突出 Z 世代占比，并与其他年龄段对比 |
| `survey.xlsx` | 问卷数据（Excel） |
